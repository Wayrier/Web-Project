# Web Portfolio

[![Pages](https://github.com/Wayrier/Web-Project/actions/workflows/pages/pages-build-deployment/badge.svg)](https://github.com/Wayrier/Web-Project/actions/workflows/pages/pages-build-deployment)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**Live:** https://wayrier.github.io/Web-Project/

A fast, responsive portfolio built with **HTML • CSS • JavaScript**.  
Projects are loaded from `projects.json`. Includes search, tag filters, and a light/dark theme.

## ✨ Features
- Responsive layout, sticky header, smooth scrolling
- Dark/Light theme toggle (saved in localStorage)
- Search + tag filters for projects
- Data-driven via `projects.json` (no build step)

## 🗂 Structure
index.html # layout & meta
styles.css # styles + theme
script.js # search/filter/theme + smooth scroll
projects.json # projects (title, description, tags, links)
assets/ # images (e.g., avatar.jpg)
